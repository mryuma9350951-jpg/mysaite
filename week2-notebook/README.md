### 写経セルの使い方

各ステップには **写経セル（空箱）** と **模範解答セル** の2種類があります。

```
■ 写経（n）— ○○○         ← タイトルと説明（何をするか・使う知識）
┌──────────────────┐
│ # コメントのみ   │  ← 空箱セル：ここに自分でコードを書く
│ # ↓ ここに書く  │
└──────────────────┘
  ▼ 写経（模範解答）— 実際に書いてから確認しよう！
┌──────────────────┐
│ 完全なコード     │  ← 模範解答：書いた後に答え合わせ
└──────────────────┘
```

> 💡 **必ず空箱セルに自分でコードを書いてから模範解答を確認してください。**
> 写経は「読む」だけでなく「手で書く」ことで理解が深まります。

## 📁 app.py を作るためのフォルダ構成

ノートブックで理解したら、**別フォルダ** に Streamlit アプリを作ります。

```
techzero_w2/                   ← 新しく作るアプリ用フォルダ
│
├── search.py        ← Step3 の2つの関数をここに書く
│     ├── search_pages()
│     └── highlight_match()
│
├── app.py           ← 自分で書く（Step4 の骨格を参考に）
│
├── requirements.txt ← 必要なライブラリを記載
│     └── 中身：streamlit>=1.32.0
│
└── pages.json       ← week2-notebook/ からコピーする
```

### app.py を書く手順

```
① techzero_w2/ フォルダを新しく作る

② search.py を作る
   Step3 の模範解答から search_pages() と highlight_match() をコピーして貼る

③ requirements.txt を作る
   中身に「streamlit>=1.32.0」と書いて保存する

④ pages.json を week2-notebook/ からコピーする

⑤ app.py を書く  ← ここが今週のメイン！
   Step4-4「app.py を書くときの骨格」を参考にしながら自分で書く

⑥ ターミナルでライブラリをインストールする
   pip install -r requirements.txt
   （streamlit はこのタイミングでインストールされます）

⑦ Step5 の練習問題で理解を確認する

⑧ Step6 で streamlit run app.py を実行して動かす！
```

> 💡 **`app.py` の完成コードは配布していません。**
> ノートブックで学んだパーツをそのまま組み合わせれば作れます。
> 「Jupyter で動いた」→「その関数を `.py` ファイルに書き出す」、これだけです。

> 💡 **`pip install` はどのフォルダで実行してもOKです。**
> `pip` はPC全体にライブラリをインストールするので、フォルダの場所は関係ありません。

## ⚠️ よくあるエラーと対処法

### `ModuleNotFoundError: No module named 'streamlit'`

Streamlit がインストールされていません。`techzero_w2/` フォルダ内の `requirements.txt` を使ってインストールしてください。

```bash
pip install -r requirements.txt
```

---

### `FileNotFoundError: pages.json`

`pages.json` がノートブック（またはアプリ）と同じフォルダにありません。

- ノートブックのセルを実行するとき → `pages.json` を `week2-notebook/` に置く
- `streamlit run app.py` を実行するとき → `pages.json` を `techzero_w2/` にコピーする

---

### `KeyError: 'category'`

練習3を複数回実行したことで `pages.json` にデータが重複登録されている可能性があります。

**対処法：** 配布された元の `pages.json`（5件入り）に差し替えてから再実行してください。

---

### `streamlit run app.py` を実行したがブラウザが開かない

ターミナルに表示された `http://localhost:8501` を手動でブラウザに貼り付けて開いてください。

---

### セルを実行しても何も起きない / カーネルが動かない

Jupyter のメニューから **「Kernel」→「Restart & Run All」** を試してください。

## 📝 ノートブックを進めるときのコツ

**写経は手で入力してください。**
コードをコピー&ペーストするより、一文字ずつ入力することで構文が頭に入ります。

**エラーが出ても慌てないでください。**
エラーメッセージの最終行が原因です。`KeyError` / `SyntaxError` / `IndentationError` は
このREADMEの「よくあるエラー」を参照してください。

**模範解答は「答え合わせ」に使ってください。**
自分で書いた後に確認する使い方が理想です。行き詰まったときのヒントとしても活用できます。

**`pages.json` を直接開いてみてください。**
テキストエディタで開くと、自分が書いたコードによってデータが変化しているのが確認できます。
「JSONはただのテキストファイル」という感覚が身につきます。
